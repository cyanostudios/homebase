## Fält utan UI – detaljerad tabell

Följande fält har inga egna input-fält i ProductForm. De antingen byggs automatiskt, importeras från Sello, eller används inte längre.

| Fält                       | Lagring                                                                                          | Anledning utan UI                                                                                                                                  | Hur det fylls                                                                                                             | Skickas till kanaler?                                                       |
| -------------------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| **specifications**         | `channel_specific.cdon.specifications`                                                           | CDON kräver språk + sektioner + attribut. Komplex struktur.                                                                                        | Manuellt via API om behov. Identifikation → Tillverkarens artikelnummer byggs i mapparen från `products.mpn` (per språk). | Ja – CDON `specifications` (byggs från MPN + befintlig cdon.specifications) |
| **manufacturer** (sträng)  | `channel_specific.cdon.manufacturer`                                                             | CDON använder `manufacturer` som objekt `{ name }`. Homebase har `manufacturer_id` → `manufacturerName` (från lookups).                            | Mapparen använder `product.manufacturerName` (från manufacturer_id). Ej separat cdon.manufacturer.                        | Ja – CDON `manufacturer` (alltid från `product.manufacturerName`)           |
| **legacy_product_id**      | `channel_specific.fyndiq.legacy_product_id`                                                      | Fyndiq-legacy för migrering från gamla plattformar. Sällan behov.                                                                                  | Manuellt via API eller import om Fyndiq kräver det.                                                                       | Ja – Fyndiq `legacy_product_id` (om satt)                                   |
| **group_id**               | `products.group_id`                                                                              | Visas skrivskyddat i flik Detaljer (Gruppering). Sätts via Group… i produktlistan (typ Färg/Storlek/Modell + huvudprodukt) eller vid Sello-import. | Sello-import eller POST /api/products/group (Group… i listan).                                                            | Indirekt – CDON `parent_sku` (groupId), WooCommerce variable product SKU    |
| **variational_properties** | `channel_specific.cdon.variational_properties`, `channel_specific.fyndiq.variational_properties` | Anger vilka egenskaper som definierar varianter (t.ex. `["color","size"]`). Härleds från Sello `groups.type`.                                      | Vid Sello-import från `groups.type` (color/size/model).                                                                   | Ja – CDON/Fyndiq `variational_properties`                                   |

**Övriga fält utan UI:** `id`, `user_id` (system), `categories` (Sello-import, inte redigerbart i formulär). Gruppering (group_id, parent_product_id, group_variation_type) visas skrivskyddat i Detaljer; sätts via Group… i produktlistan eller Sello-import.

## Sammanfattning

- **WooCommerce gruppering:** Produkter med `group_id` och `group_variation_type` (color/size/model) exporteras som variable product (SKU = group_id) + variations (SKU = V+product.id). Uppdatering/lookup utan sparade Woo-ID: GET ?sku= för att hämta Woo id, sedan PUT/POST.
- **notes:** Internt för Homebase – visas och sparas i Attribut, skickas inte till någon kanal.
- **Fält utan UI:** Se tabellen ovan (specifications, manufacturer, legacy_product_id, group_id, variational_properties).
- **source_created_at, quantity_sold, last_sold_at:** Skrivs vid Sello-import. Visas i Statistik-fliken, inte redigerbara.
- **unique_selling_points = bulletpoints:** CDON:s namn. Fylls i i Bulletpoints (flik Texter). Exportas till CDON.
- **kn_number:** Egen kolumn. Tullklassificering. Flik Detaljer.
- **classifications:** CDON CONDITION (NEW/USED/REFURB). Byggs från condition (Ny→NEW, Used→USED, Refurbed→REFURB).
